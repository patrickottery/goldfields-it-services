# Images folder

## About page photo

To show your photo on the **About** page:

1. Save your photo in this folder (`images/`) with the filename **`about-photo.jpg`**.
2. Supported formats: `.jpg`, `.jpeg`, or `.png`.  
   - If you use PNG, rename the file to `about-photo.png` and update `about.html`: change `images/about-photo.jpg` to `images/about-photo.png`.

**Tips:**
- A square or nearly square photo works best (it’s displayed in a square frame).
- Good size: at least 400×400 pixels. Larger is fine; the site will scale it down.
- A clear, friendly headshot or professional photo works well.

No need to change anything else—the About page will pick up the image automatically.
